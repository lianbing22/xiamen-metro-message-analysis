# Task: 文件管理模块开发

## Description
开发文件上传、解析和存储功能模块，支持Excel、CSV等格式的报文文件上传，实现文件的批量处理和异步解析。

## Acceptance Criteria
- [ ] 实现拖拽上传组件，支持多文件同时上传
- [ ] 文件格式验证功能（支持.xlsx、.xls、.csv）
- [ ] 文件大小限制检查（最大100MB）
- [ ] Excel/CSV解析服务，提取报文数据
- [ ] MinIO对象存储集成
- [ ] 文件上传进度显示和状态管理
- [ ] 文件列表查询和下载功能

## Technical Details
- 前端：使用vue-upload-component或element-upload
- 后端：Spring Boot MultipartFile处理
- 解析：Apache POI (Excel)、OpenCSV (CSV)
- 存储：MinIO或本地文件系统
- 异步处理：RabbitMQ队列
